# Physical AI & Humanoid Robotics – Chapter Planning Skill

## When to Use This Skill

- User wants to plan a chapter for a Physical AI / humanoid robotics book
- User asks for chapter outline, title ideas, or intro for a robotics/AI topic
- User needs help structuring technical content (theory + code + examples)

## How This Skill Works

1. **Clarify chapter topic**: Understand the specific Physical AI / robotics concept
2. **Identify core concepts**: Extract 3–4 key ideas that chapter will teach
3. **Create outline**: Break chapter into 7–10 sections (theory, practice, examples)
4. **Suggest titles**: Provide 5 strong chapter title options
5. **Draft introduction**: Write 1–2 paragraph hook for technical readers

## Output Format

Provide:

- **Chapter Topic**: 1 line
- **Core Concepts**: Bullet list of 3–4 concepts
- **Target Audience**: 1–2 sentences
- **Chapter Outline**: Numbered list (7–10 sections)
- **Chapter Title Options**: 5 variations
- **Introduction Draft**: 1–2 paragraphs
